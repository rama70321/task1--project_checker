# Password Strength Checker 🔐

## Project Description

This project is developed as part of the DecodeLabs Cyber Security Industrial Training (Project 1).

The Password Strength Checker is a Python program that checks whether a password is **Weak**, **Medium**, or **Strong** based on basic security rules.

## Features

- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special symbols
- Displays password strength (Weak / Medium / Strong)

## Technologies Used

- Python 3
- VS Code

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python password_checker.py
```

4. Enter a password when prompted.
5. View the password strength result.

## Example Output

```
Enter your password: Pass@123

Password Analysis
---------------------------
Length: 8
Contains Uppercase: True
Contains Lowercase: True
Contains Number: True
Contains Symbol: True

Password Strength: STRONG
```
